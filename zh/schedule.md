# 暂定课程安排

{:.no_toc}

- 本安排在学期中可能会有小幅调整，但考试日期不会变化。
- slides 通常会在每次 lecture 前上传到课程[主页]({{ '/zh/' | relative_url }})。lecture 内容本身可能与教材差异较大，因此建议现场参加或课后及时看录像。
- guest lecture 不纳入考试范围，但会成为 extra credit 活动的重点内容。
- 某些主题可能会持续数周。

| 周次 | 主题 |
|------|------|
| 1-2  | 基础：深度学习、计算图、自动微分、ML 框架 |
| 3    | GPU、CUDA、Collective Communication |
| 4    | 图与内存优化 |
| 4    | Guest lecture：ML 编译器 |
| 5    | 数据并行、模型并行、自动并行化 |
| 6    | Transformers、LLMs、MoE |
| 6    | Guest lecture：LLM 预训练与开放科学 |
| 7    | LLM 训练：Flash Attention、量化 |
| 8    | LLM 推理与服务：Paged Attention、Continuous Batching、Speculative Decoding |
| 9    | Guest lecture：快速推理 |
| 9    | Scaling Law、test-time compute、reasoning |
| 10   | LLM + X（X = RAG、搜索、多模态等） |
| 10   | Guest lecture：LLM、tool use 与 agents |
| 10   | 期末复习 |
| 11   | Final exam |
